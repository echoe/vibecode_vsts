A six oscillator synthesizer that you can swap any of the oscillators into filters with.
Options:
-Oscillator types: Sine, Triangle, Saw, Pulse, Additive
-Filter types: Lowpass, Highpass, Bipole, Comb

This is probably 95% vibe coded but I did yell at the AI a fair bit and handcoded some of the for loops to make sure e.g. that I could scale the oscillators (I started with 2 and built up). I haven't built this on Windows or MacOS either. have fun :)
