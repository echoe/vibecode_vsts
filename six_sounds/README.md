A six oscillator synthesizer that you can swap any of the oscillators into filters with.


Synth options:


\-Oscillator types: Sine, Triangle, Saw, Pulse, Additive


\-Filter types: Lowpass, Highpass, Bipole, Comb



Pictures are provided in the pictures folder. Builds are provided in the vst3\_builds folder.



This is probably 99% AI/vibe coded but I did yell at the AI a fair bit, and interrupted/guided where I thought it was going wrong. It builds on my machine (on Windows, MacOS, and Linux).
